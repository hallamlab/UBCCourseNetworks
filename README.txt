==== UBCCourseNetworks ====

This is a web app created at the Hallam Lab in the UBC Department of Microbiology and Immunology.  The live version can be found at http://coursemapper.microbiology.ubc.ca/UBCNetworks. It allows the user to visualize UBC courses as networks connected by their prerequisites and corequisites.

The source code may be used for non-commercial purposes, as long as the Hallam Lab is acknowledged. See the documentation at https://github.com/hallamlab/UBCCourseNetworks/wiki/Documentation.
